# نقشه راه پروژه هفته‌یار (Hafte-Yar Project Roadmap)

این سند مراحل توسعه و وضعیت فعلی پروژه هفته‌یار را نشان می‌دهد.

---

## وضعیت فعلی (Current Stage)
* [x] **پایه بک‌اند (Core Backend)**: طراحی و پیاده‌سازی مدل‌های دیتابیس (کاربران، تیم‌ها، پروژه‌ها، لیست‌ها، تسک‌ها) و سرویس‌های پایه با FastAPI.
* [x] **احراز هویت OTP**: پیاده‌سازی سیستم ارسال و تایید کد تایید (SMS/Console).
* [x] **معماری وب‌هوک پیام‌رسان‌ها**: طراحی آداپتورهای تلگرام و بله و ایجاد اندپوینت دریافت وب‌هوک.
* [/] **تکمیل قابلیت‌های وب‌هوک (در حال انجام)**: پیاده‌سازی مفسر دستورات متنی (Command Parser) در ربات‌ها برای ایجاد تیم، دعوت اعضا، ساخت تسک و بروزرسانی وضعیت تسک‌ها.

---

## مراحل آتی توسعه (Future Phases)

### فاز ۱: تکمیل و تثبیت بک‌اند و پیام‌رسان‌ها (کوتاه‌مدت)
۱. **تکمیل مفسر ربات (message_service.py)**:
   * پیاده‌سازی دستورات تیمی (`/create_team` , `/invite` , `/teams` , `/members`).
   * پیاده‌سازی دستورات تسک (`/tasks` , `/team_tasks` , `/create_task` , `/update_task`).
۲. **رفع نواقص و تست بک‌اند**:
   * ساخت خودکار پروژه‌ها و لیست‌های پیش‌فرض هنگام ایجاد تیم برای سهولت استفاده در ربات‌ها.
   * تست عملکرد وب‌هوک‌ها با استفاده از کلاینت‌های شبیه‌ساز (Mock Test).

### فاز ۲: ساخت پنل وب (وب‌سایت هفته‌یار)
۱. **فرانت‌اند وب (Web Frontend)**:
   * طراحی و ساخت رابط کاربری با React / Next.js / Vite.
   * طراحی داشبورد مدیریت تسک‌ها (تخته کانبان - Kanban Board).
۲. **اتصال به بک‌اند (Integration)**:
   * اتصال پنل وب به APIهای بک‌اند.
   * پیاده‌سازی سیستم مدیریت ورود یکپارچه با توکن صادر شده از ربات‌های پیام‌رسان (Login Callbacks).

### فاز ۳: پیاده‌سازی عامل هوش مصنوعی (AI Agent Middleware)
۱. **طراحی لایه میان‌افزار (Middleware Layer)**:
   * ایجاد سرویس دریافت فایل‌های صوتی، تصویری و ویدیویی.
۲. **پردازش زبان طبیعی و تصویر (NLP & Computer Vision)**:
   * استفاده از مدل‌های هوش مصنوعی (مانند Whisper برای صوت و GPT-4o / Gemini برای تصویر/ویدیو) جهت استخراج متادیتای تسک‌ها (عنوان، توضیحات، مسئول، تاریخ سررسید).
۳. **ثبت خودکار تسک (Auto-Task Creation)**:
   * نگاشت اطلاعات استخراج شده به API بک‌اند هفته‌یار و ثبت و تخصیص خودکار تسک‌ها.
